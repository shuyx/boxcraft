常见问题

没有 AI Box 设备怎么开始？ ▼
报名不需要设备。通过筛选后我们会免费寄送 AI Box。前期可使用 SDK 模拟环境开发。
可以使用云端 API 吗？ ▼
可以辅助使用，但核心功能须在端侧运行。端侧能力是评审加分项。
可以用 AI 辅助编程吗？ ▼
鼓励使用！需在 README 中披露工具名称，代码须可解释、可复现。
作品 IP 归谁？ ▼
归参赛团队所有。不强制开源。赛后 60 天内主办方享有优先合作权。
奖金怎么发？ ▼
税前金额，代扣 20% 个人所得税后银行转账，颁奖后 30 天内到账。
决赛需要到现场吗？ ▼
是的。决赛在上海虹桥国展中心金色大厅（机器人年会主会场）进行现场路演。
