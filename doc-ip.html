📜 IP 与知识产权

📌 版本 v0.4 📅 2026-03-05

一、知识产权归属

条款 说明
作品 IP 归属 作品知识产权（含代码/文档/素材等）默认归参赛团队所有
赛事授权 主办/赞助方获得非独占、合理期限的使用授权，仅用于：评审、展示、赛事传播
开源要求 不强制开源；鼓励但不要求公开代码
最小交付 即使不开源，也需提交可复现 Demo 包 + 作品说明 + 演示视频

二、主办方优先权益

权益 说明
优先投资/合作窗口 赛后 60 天内，主办/赞助方享有优先投资、PoC 合作、许可/收购谈判的权利
优先沟通 窗口期内，参赛团队如收到第三方意向，需优先通知主办方
窗口期后 60 天结束后，团队自由处置，不设任何限制
60 天足够完成初步对接，且对参赛者友好。

三、仓库公开策略

决策：赛后统一公开
  • 开发期：默认私有（评委/导师可见）
  • 半决赛/决赛：仍保持私有
  • 赛后：统一转为公开（获奖公示后 1-2 周内）
  • 不愿公开的团队：可申请保持私有，但需提交可复现 Demo 包供展示

理由：赛后公开更防抄袭、减少参赛者压力，且合规检查可在公开前完成。
