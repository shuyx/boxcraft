快速上手

三步开始你的 Agent 开发之旅

01

环境准备

连接 AI Box，安装 SDK，配置开发环境

$ pip install aibox-sdk
02

Hello Agent

运行你的第一个端侧 Agent 示例

$ aibox run hello-agent
03

构建 & 部署

将 Agent 打包并部署到 AI Box

$ aibox deploy --target box
